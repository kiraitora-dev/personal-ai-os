# Personal AI OS

Personal AI OS is a personal AI agent designed to run 24/7 on Windows 11.

## Vision
- Local-first
- Modular agent architecture
- Persistent memory
- Task automation
- Voice and chat interfaces
- Extensible tools

## Planned architecture
```
app/
core/
agents/
memory/
automation/
config/
docs/
```

This repository is being developed incrementally with small, traceable commits.
